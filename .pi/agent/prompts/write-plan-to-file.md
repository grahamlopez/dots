$@

Please write this plan out to a file in docs/plans with the implementation steps clearly marked and with enough detail so that they can be independently implemented by separate agents.

To complete a robust and successful plan, reread the document after completion and check:
- Ensure that each step is independent and logically sequenced, so that no steps require changes that are specified by later steps.
- Look for any contradictions in the instructions or statements that could be confusing for future agents.
- Consider the amount of work required by each step and that it is scoped small enough for very efficient sub-agent implementation.
- There doesn't need to be a separate testing step; testing should naturally be a part of each step's implementation process.
